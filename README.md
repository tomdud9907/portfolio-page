# Portfolio Page

This is a personal portfolio page generated with **Codex** for fun and to explore the latest AI development tools and workflows.

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS

## Getting started

### 1) Install dependencies

```bash
npm install
```

### 2) Run development server

```bash
npm run dev
```

The app will be available locally (typically at `http://localhost:5173`).

### 3) Build for production

```bash
npm run build
```

### 4) Preview production build

```bash
npm run preview
```

## Notes

- The project includes multiple portfolio sections (Home, Experience, Projects, About).
- Content and styles are intentionally easy to edit so this can be a practical starter template.
